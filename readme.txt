Настройка возможности писать пользователям в определенные категории. 

== Description ==
По умолчанию Авторы, Модераторы и Администраторы могут писать везде. Блоггеры нигде пока им не будет указано куда им можно оставлять сообщения. Исключение для Блоггеров составляет категория «Без Категории».

= How it works: =
Плагин использует низкоуровневые фильтры Wordpress для вывода категорий с ограничениями на страницах админ-панели.
Настройки пользователей хранятся в мета-полях.
При устновке плагина создается роль Блоггер. При удалении она удаляется, при этом вывод всех категорий будет работать как до установки плагина, но сохраненные мета-поля пользователей остнутся.
Фронт-енд не затронут.

== Installation ==
Плагин может быть установлен автоматически через Плагин-панель вашей админ-панели.

= To install manually instead: =
1. Заргузить `role.zip` в папку `/wp-content/plugins/`
1. Распаковать `role.zip` в папку `/wp-content/plugins/`
1. Активировать плагин через 'Plugins' меню в WordPress
